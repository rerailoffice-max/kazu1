## 1. 事故概要

### 基本情報

| 項目 | 内容 |
|------|------|
| **事故発生日時** | 2022年（令和4年）4月23日（土）13:00頃〜14:00頃 |
| **事故発生場所** | 北海道斜里郡斜里町 知床半島西海岸沖「カシュニの滝」付近（北緯44度10分、東経145度03分付近） |
| **事故船舶** | 観光船「KAZU I（カズ ワン）」 |
| **運航会社** | 有限会社知床遊覧船 |
| **乗船者数** | 26名（乗客24名＋乗員2名） |
| **被害** | 死亡20名、行方不明6名（全員犠牲） |
| **事故種類** | 船内浸水による沈没 |

### 事故の位置づけ
- 事故当日は知床遊覧船の2022年シーズン運航初日
- 他社（5社中4社）はゴールデンウィーク（4月29日頃）から運航開始予定だったため、当日運航していたのは同社のみ

## 6. 事故原因（運輸安全委員会報告）

### 6.1 直接的原因

| 原因 | 詳細 |
|------|------|
| **ハッチからの浸水** | 船首甲板部ハッチの蓋が確実に閉鎖できる状態ではなく、船体の動揺により開いて浸水 |
| **浸水の拡大** | 船体下部の隔壁に開口部があり、浸水が後方に拡大 |
| **エンジン停止** | 海水が船内の電子部品をショートさせ、エンジンが停止した可能性 |
| **沈没** | 短時間で沈没に至る |

### 6.2 背景要因

#### 運航会社の問題

| 問題点 | 詳細 |
|--------|------|
| **安全管理体制の欠如** | 実質的な運航管理を行わず、経験が浅い船長に判断を任せていた |
| **運航基準の不遵守** | 出航中止基準（風速8m/s以上、波高1m以上）を超える予報が出ていたにもかかわらず出航 |
| **通信設備の不備** | 無線機のアンテナが壊れているなど、常時接続が困難な状態 |
| **救命設備の不足** | 低水温環境（水温3度強）に対して不十分な救命設備 |
| **運航管理者の不在** | 桂田社長は2021年シーズン中、定点連絡業務にほとんど従事せず。事故当日も不在 |

#### 国・検査機関の問題

| 機関 | 問題点 |
|------|--------|
| **北海道運輸局** | 2021年の事故後に特別監査を実施したが、ヒアリングのみで「安全意識が向上された」と表面的な評価 |
| **日本小型船舶検査機構（JCI）** | 事故3日前の検査でハッチの蓋の開閉状況を実際に確認せず、目視のみで判断 |

### 6.3 運輸安全委員会の結論（2023年9月7日最終報告書）

> 運航会社、国、検査機関の3者とも安全確保への取り組みが不十分だったことが事故を招いた

**出典**: [運輸安全委員会報告書](https://jtsb.mlit.go.jp/jtsb/ship/detail.php?id=14055)、[日本経済新聞](https://www.nikkei.com/article/DGXZQOUE05C0X0V00C23A9000000/)、[北海道新聞](https://www.hokkaido-np.co.jp/article/905149/)
