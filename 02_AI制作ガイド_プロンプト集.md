# 「KAZU I - 72時間」AI映像制作ガイド

## 使用AIツール

### 画像生成
- **Midjourney**（メイン）：静止画・キービジュアル生成
- **Ideogram**（サブ）：テキスト入り画像が必要な場合

### 動画生成
- **Kling AI**：リアルな人物・動きのある映像
- **Runway Gen-3**：シネマティックな映像、カメラワーク
- **Luma Dream Machine**：環境・風景の動画化

### 補足ツール
- **Topaz Video AI**：アップスケール・フレーム補間
- **DaVinci Resolve**：編集・カラーグレーディング
- **Adobe Premiere Pro**：最終編集

---

## 制作ワークフロー

```
1. Midjourneyで静止画を生成
2. 静止画をKling/Runway/Lumaで動画化
3. Topazでアップスケール・補間
4. DaVinci/Premiereで編集
```

---

## 全体のビジュアルスタイル

### 【A視点】乗客カメラ風
- アスペクト比：16:9（横長）
- 画質：やや粗い、手持ちカメラ風
- 色調：暖色系→寒色系へ変化
- 特徴：手ブレ、レンズフレア、ノイズ

### 【B視点】俯瞰・ドキュメンタリー風
- アスペクト比：16:9（横長）
- 画質：クリア、プロフェッショナル
- 色調：ニュートラル〜寒色系
- 特徴：安定したカメラ、ドローン風

## 制作上の注意事項

### 倫理的配慮
1. 実在の被害者の顔は一切生成しない
2. 遺体の直接的な描写は避ける
3. 過度にセンセーショナルな演出は控える
4. 遺族への敬意を常に意識する

### 技術的注意
1. 【A視点】は意図的に画質を落とす（ノイズ追加）
2. 【B視点】はプロフェッショナルな画質を維持
3. 色調は前半（暖色）→後半（寒色）へ変化
4. 音楽は使用しない（環境音のみ）

### Midjourney設定
- バージョン：v6.1推奨
- スタイル：--style raw（リアリズム重視）
- アスペクト比：--ar 16:9
- 品質：--q 2（高品質）
